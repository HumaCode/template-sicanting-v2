Pertanyaan Umum (FAQ)

  1. Buka halaman login SICANTING
  2. Masukkan email dan password yang telah terdaftar
  3. Klik tombol "Masuk"
  4. Jika lupa password, klik "Lupa Password?" untuk reset
Pastikan kredensial Anda sudah terdaftar oleh administrator

  1. Masuk ke menu Pengguna
  2. Klik tombol "Tambah Pengguna"
  3. Isi semua data yang diperlukan (nama, email, role, dll)
  4. Klik tombol "Simpan"
Fitur ini hanya tersedia untuk role Administrator

Pokja (Kelompok Kerja) adalah pembagian bidang tugas dalam organisasi PKK.

Cara mengelola:

  1. Buka menu Pokja
  2. Untuk menambah: klik "Tambah Pokja"
  3. Untuk mengedit: klik icon pensil pada card Pokja
  4. Untuk menghapus: klik icon tempat sampah

  1. Masuk ke menu Pengaturan > Akses Role
  2. Klik tombol "Tambah Role"
  3. Pilih role yang ingin diatur
  4. Centang permission yang diinginkan untuk setiap modul
  5. Klik "Simpan"

Permission yang tersedia:

  • Read - Melihat data
  • Create - Menambah data baru
  • Update - Mengubah data
  • Delete - Menghapus data sendiri
  • Delete Any - Menghapus data siapapun

  1. Masuk ke Pengaturan Sistem
  2. Pilih tab "Backup"
  3. Klik tombol "Backup Sekarang"
  4. Tunggu proses selesai
  5. File backup akan tersimpan otomatis
Backup otomatis dapat dikonfigurasi untuk berjalan setiap hari
