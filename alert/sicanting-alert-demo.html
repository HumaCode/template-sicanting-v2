SICANTING Custom Alert

Library notifikasi & alert yang selaras dengan tema SICANTING

Cara Install

Tambahkan dua file berikut ke setiap halaman SICANTING

<link rel="stylesheet" href="sicanting-alert.css">
<script src="sicanting-alert.js"></script>
1. Toast Notification

Notifikasi kecil yang muncul di pojok layar. Otomatis hilang setelah beberapa detik.

Pilih posisi toast:

// Toast sederhana
SCA.toast({
    type    : 'success',       // primary | success | warning | danger | info | dark | light
    title   : 'Berhasil!',
    message : 'Data berhasil disimpan.',
    duration: 4000,            // milidetik, 0 = tidak hilang otomatis
    position: 'top-right',    // top-right | top-left | top-center | bottom-*
});

// Shorthand
SCA.success('Berhasil!', 'Data tersimpan.', true);  // parameter ke-3 true = toast
SCA.error('Gagal!', 'Terjadi error.', true);
SCA.warning('Peringatan!', 'Sesi hampir berakhir.', true);
2. Alert Banner (Inline)

Alert yang tampil di dalam halaman, bisa di-dismiss oleh user.

Alert akan muncul di sini

// Alert inline dengan target selector
SCA.alert({
    type    : 'warning',
    title   : 'Perhatian!',
    message : 'Ada data yang belum diisi.',
    dismissible: true,   // tampilkan tombol close
}, '#containerSaya');   // CSS selector atau DOM element
3. Dialog / Modal Konfirmasi

Modal konfirmasi dengan Promise. Gunakan .then() untuk menangani pilihan user.

// Dialog dengan Promise
SCA.dialog({
    type        : 'danger',
    title       : 'Hapus Data?',
    message     : 'Data tidak dapat dikembalikan.',
    confirmText : 'Ya, Hapus',
    cancelText  : 'Batal',
    showCancel  : true,
}).then(result => {
    if (result) {
        // User klik konfirmasi
        SCA.toast({ type: 'success', title: 'Terhapus!' });
    }
});

// Shorthand delete
SCA.deleteConfirm('Nama Item').then(result => { ... });

// Shorthand confirm
SCA.confirm('Judul', 'Pesan').then(result => { ... });
4. Loading Dialog

Tampilkan loading saat memproses data, lalu tutup dan tampilkan hasil.

// Tampilkan loading
SCA.loading({
    title  : 'Menyimpan...',
    message: 'Mohon tunggu sebentar'
});

// Simulasi proses async
setTimeout(() => {
    SCA.close();   // tutup loading
    SCA.toast({ type: 'success', title: 'Berhasil!' });
}, 2000);

// Dengan fetch/axios
SCA.loading({ title: 'Loading...' });
fetch('/api/data')
    .then(res => res.json())
    .then(data => {
        SCA.close();
        SCA.success('Berhasil!', 'Data dimuat.');
    });
5. Shorthand Methods

Method cepat untuk penggunaan umum. Parameter ke-3 true = tampil sebagai toast.

// Semua shorthand method:

SCA.success(title, message, isToast);   // isToast default = false (dialog)
SCA.error(title, message, isToast);
SCA.warning(title, message, isToast);
SCA.info(title, message, isToast);
SCA.confirm(title, message, opts);         // selalu dialog, return Promise
SCA.deleteConfirm(itemName);               // konfirmasi hapus, return Promise
SCA.loading({ title, message });
SCA.close();                               // tutup dialog/loading aktif
